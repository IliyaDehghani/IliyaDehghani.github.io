About Me

I am a cybersecurity student with hands-on experience in penetration testing, threat detection, digital forensics, and secure system design. My background includes conducting OS and web application penetration tests, building centralized monitoring pipelines, investigating breaches using forensic tools, hardening systems to industry benchmarks, and deploying complex virtualized environments. I also develop automation scripts in Python and Bash, and leverage Ansible for efficient system provisioning and configuration management.

Experience

  • Conducted OS and web application penetration testing using industry-standard tools (Kali Linux, Burp Suite, Metasploit); identified vulnerabilities aligned with OWASP Top 10.
  • Built a centralized monitoring pipeline using Wazuh, Splunk, and log analytics to detect suspicious activity, generate alerts, and perform threat correlation.
  • Investigated simulated breaches using Autopsy; recovered forensic artifacts, built timelines, and documented findings following incident response procedures.
  • Performed OS hardening on Linux systems using CIS Benchmarks and custom scripts to reduce attack surface and improve security posture.
  • Deployed a virtualized lab using Proxmox, Sophos Firewall, and Docker to simulate enterprise environments; implemented remote access, VLANs, and traffic segmentation.
  • Leveraged Python and Bash to develop automation scripts for system monitoring, log parsing, and security task orchestration.
  • Automated system provisioning and configuration management using Ansible, improving consistency and efficiency.
